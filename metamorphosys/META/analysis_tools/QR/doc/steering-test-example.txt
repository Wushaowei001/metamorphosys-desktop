To run the example
0) Copy and unzip the SteeringTest.zip file from the installed QRM/test-models directory to a temporary directory
1) we need to open the SteeringTest.mga file.
2) In the GME browser go to SteeringSystem->TestBenches->SteeringSystem->SteeringSystem and double click on it
3) Click on the CyPhy Modelica Interpreter icon (M) on the toolbar
4) (optional add libraries - should not have to do this)
5) when the job manager starts click run
6) When the status of the job changes from "RunningLocal" to "Succeeded", open the browser window
7) Reload the browser page and "ctrl+click" on the text for "Design Space Analyzer" You should see a single requirement "Limit1" with a "Yellow" indicator.
